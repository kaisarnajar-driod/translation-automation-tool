Features

Everything you need for localization

From a single CLI command to a full web dashboard — Transync handles the entire translation pipeline.

🔌

Git-Native Workflow

Provide a repo URL from GitHub, Bitbucket, GitLab, or any host. Transync clones, pulls, translates, commits, and pushes automatically.

🌐

Google Translate (Free)

Uses Google's Neural Machine Translation via deep-translator. No API key, no billing, no setup friction.

🔍

Smart Diff Detection

Only translates new or modified strings. Automatically removes deleted keys from all language files. Incremental, snapshot-based.

🛡

Placeholder Safety

Validates that %s, %1$d, HTML tags, and format specifiers survive translation. Falls back to source text on corruption.

🖌

Web Dashboard

Dark-themed single-page UI to add projects, trigger syncs, view results, and manage languages — no terminal needed.

⌨

Beautiful CLI

Rich-powered tables, progress bars, colored output, and dry-run mode. Everything you need from the command line.

Quick Start

Up and running in 2 minutes

Install once, add your repos, and start syncing.

1 Install

# Clone & install
git clone git@github.com:kaisarnajar-driod/translation-automation-tool.git
cd translation-automation-tool
pip install -e .

2 Configure

# Generate config file
transync init

# No API key needed!
# Uses Google Translate free tier

3 Add a project

transync add my-app \
  https://github.com/org/app.git \
  --strings-path values/strings.xml \
  --languages hi,ar,fr,es

4 Sync!

# CLI
transync sync my-app

# Or launch the web UI
transync serve
