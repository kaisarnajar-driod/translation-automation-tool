Features

Everything you need for localization

From a single CLI command to a full web dashboard — Transync handles the entire translation pipeline.

🔌

Docker-Ready

One command to build and run. No Python, no pip, no virtual environments needed on your machine — just Docker.

🌐

Google Translate (Free)

Uses Google's Neural Machine Translation via deep-translator. No API key, no billing, no setup friction.

🔍

Git-Native Workflow

Provide a repo URL from GitHub, Bitbucket, GitLab, or any host. Transync clones, pulls, translates, commits, and pushes automatically.

🛡

Placeholder Safety

Validates that %s, %1$d, HTML tags, and format specifiers survive translation. Falls back to source text on corruption.

🖌

Web Dashboard

Dark-themed single-page UI to add projects, trigger syncs, view results, and manage languages — no terminal needed.

🔍

Smart Diff Detection

Only translates new or modified strings. Removes deleted keys from all language files. Incremental, snapshot-based.

Terminal — Start Transync with Docker
# Clone and build
$ git clone git@github.com:kaisarnajar-driod/translation-automation-tool.git
$ cd translation-automation-tool
$ docker build -t transync .

Successfully built transync:latest

# Run the container
$ docker run -d --name transync -p 8090:8090 \
  -v transync-data:/root/.transync \
  -v ~/.ssh:/root/.ssh:ro \
  transync

Quick Start

Up and running in 2 minutes

Just Docker. No Python, no pip, no virtual environments.

1 Clone & Build

git clone git@github.com:kaisarnajar-driod/translation-automation-tool.git
cd translation-automation-tool
docker build -t transync .

2 Run

docker run -d --name transync \
  -p 8090:8090 \
  -v transync-data:/root/.transync \
  -v ~/.ssh:/root/.ssh:ro transync

3 Add a project

# Open http://localhost:8090
# Enter SSH URL, strings path, languages
git@github.com:org/app.git
app/src/main/res/values/strings.xml

4 Sync!

# Click "Sync" in the web UI
# Transync pulls, translates,
# commits, and pushes — done.
No API key needed.
